# new_prusa_mmu_hardware
原mmu2升级版主板，加入串口电平转换，dcdc独立供电系统
可更换的电机驱动，比原版更低的价格，兼容12~24V电平，
使用PNP常开接近传感器，板载串口，传感器电平转换电路
