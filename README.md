# new_prusa_mmu_hardware

原mmu2升级版主板，加入串口电平转换，dcdc独立供电。可更换的电机驱动，比原版更低的价格，兼容12~24V电平，
使用PNP常开接近传感器，板载串口，传感器电平转换电路.分体的按键小板方便打印机布线。

直插mks主板，兼容更多打印机。

确保你的打印机主板拥有一个空闲的串口和两个gpio,使用marlin固件。voron 和i3 结构可以100%支持mmu换料器。
