# new_prusa_mmu_hardware
平替原版mmu2主板,单板只需要约100(CNY)\
  PCB_V4.1|参数|IC
  --------|---|-------
  电源|12~24V|MP1584
  串口|5V/3.3V|TXS0102
  FINDA传感器|12~24V|三线PNP常开
电机驱动|/|TMC2130

直插mks主板，兼容更多打印机。\
无论你的打印机是否来自prusa生产，只要主板拥有一个空闲的串口和两个gpio,使用marlin固件。都可以使用mmu换料器。
